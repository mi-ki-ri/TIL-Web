    • CircleCIにプロジェクトを追加

      • GitHubにてリポジトリを作る
      • ローカルにクローン
        • ローカルに .circleci というフォルダを作る。
        • .circleci の中に config.yml ファイルを作る。
      • CircleCIにてサインアップ
        • ADD PROJECT でさっき作ったリポジトリを選ぶ。
        • Start Building
      • とりあえずtestが動く
